Problem Statement


Build a TIC TAC TOE Game. using HTML CSS and Javascript.

Assumption:


  1. How many players Involved?
  2. What symbol should Show for User? example X or O

Pseudocode:


For Game Board Design


  • Title board
  • Game board(3*3)
  • Result board to show Result(It should show on top of game board)
  • Button to Start game

Play Game(Javascript Part)


  • Set X and O as the first and second player
  • Attach onclick event to each cell
  • Every click write X/O on the cell based on player's turn
  • Once write dont let the player click again on the same cell
  • Check if the player won or not? check if any of the rows, column, or diagonal has 3 same values
  • Show the Result Board on top of the Game Board
  • Reset the game when the player wor or draws. Hide the result Board
